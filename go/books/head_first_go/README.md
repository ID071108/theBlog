# Head First Go

## 概述
Go是一种注重简单性和速度的编程语言。它比其他语言都简单，所以学起来更快。它可以让你利用当今多核计算机处理器的能力，使你的程序运行得更快。

## 创造的背景:
1. 快速编译 
2. 不太笨重的代码
3. 自动释放未使用的内存（垃圾收集）
4. 易于编写同时执行多个操作的软件（并发）
5. 很好地支持多核处理器
